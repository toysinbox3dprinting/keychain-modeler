Legacy manual build steps are deprecated.

Use the scripted workflow:
1) npm ci
2) npm run build:web
3) APP_BASE_PATH=/your/subpath npm run build:prod
4) APP_BASE_PATH=/your/subpath npm run build:artifact

Notes:
- Base path is driven by REACT_APP_BASE_PATH / APP_BASE_PATH.
- Static asset path is emitted by Vite using the configured base path.
- build:artifact packages build output into client.zip under the root folder name "client".
- No source files are modified by any build command.
- Geometry adapter bundles are generated/checked with:
  - npm run generate:adapters
  - npm run verify:adapters
- Desktop build/package commands:
  - npm run build:desktop
  - npm run pack:desktop:macos
  - npm run pack:desktop:windows
  - npm run build:desktop:macos
  - npm run build:desktop:windows
  - Packaging is explicit by target OS (no auto-detect desktop package command).
  - Desktop artifacts are unsigned by default; signing/notarization must be added in release automation.
